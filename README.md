# Projeto_AstroTECH---Loja-Virtual
AstroTECH é uma loja virutal moderna desenvolvida para simular de forma completa o processo de compra em um e-commerce real. O projeto integra funcionalidades essenciais de uma plataforma de vendas online com foco em organização, responsividade e boa experiência de uso.

Entre as principais funcionalidades implementadas estão:

- Sistema de login e cadastro, garantindo que apenas usuários autenticados possam avançar para a etapa de pagamento.
- Preenchimento automático de dados do usuário (como nome completo) nos formuláros de compra, a partir das informações salvas no cadastro.
- Carrinho de compras dinâmico, com cálculo automárico do valor total utilizando armazenamento local (localStorage).
- Página de finalização de compra com validações, feedback visual e mensagem de confirmação, incluindo aviso de envio de notificação por e-mail.
- Layout responsivo que se adapta de forma automática para desktop (PC), mantendo o padrão visual.
- Organização de imagens e elementos visuais padronizados, garantindo consistência e apresentação "profissional".

O AstroTECH foi construído utilizando HTML, CSS e JavaScript puro, aplicando conceitos de manipulação do DOM, eventos, máscaras de input, armazenamento local e boas práticas de UI/UX. É um projeto ideal para demonstrar domínio dos fundamentos do desenvolvimento web e a capacidade de criar soluções funcionais do zero.
